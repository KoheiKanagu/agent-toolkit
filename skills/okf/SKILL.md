# OKF Skill

Use this skill when you need to create or review an Open Knowledge Format (OKF) bundle.
The canonical reference is `skills/okf/references/SPEC.md`, which is a vendored copy of the upstream OKF specification.
When working with a bundle, follow the structure and conventions in that document for concept files, YAML frontmatter, cross-links, indexes, logs, and citations.
